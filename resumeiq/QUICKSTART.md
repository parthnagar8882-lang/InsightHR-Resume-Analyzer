# ⚡ QuickStart - InsightHR Nexus in 5 Minutes

**Get up and running with InsightHR in just 5 minutes!**

---

## 🚀 Start Now (Windows)

```powershell
cd d:\py_project\resumeiq

# Create & activate environment
python -m venv venv
venv\Scripts\activate

# Install & run
pip install -r requirements.txt
python app.py
```

✅ **Running!** → Open http://127.0.0.1:5000

---

## 🚀 Start Now (macOS/Linux)

```bash
cd ~/py_project/resumeiq

# Create & activate environment
python3 -m venv venv
source venv/bin/activate

# Install & run
pip install -r requirements.txt
python app.py
```

✅ **Running!** → Open http://127.0.0.1:5000

## ❓ Troubleshooting

**Port 5000 in use?**
```powershell
# Edit app.py last line:
app.run(debug=True, port=5001)  # Use 5001
```

**Module not found?**
```powershell
# Ensure venv activated (should see (venv) prefix)
venv\Scripts\activate
pip install -r requirements.txt
```

**Database error?**
```powershell
# Delete old database
del database.db
# Let app recreate it
python app.py
```
